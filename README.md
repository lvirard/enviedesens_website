# enviedesens_website
Site web réalisé pour l'auto-entreprise Envie de Sens, spécialisée dans les bilans de compétences sur la région de Béziers.

# Collaboration avec Envie de Sens pour ajout de contenu en mode CI/CD :
Objectif : 
Hébérgement sur OVH - mise en place d'un pipeline pour tester la validité des fichiers avant push auto sur OVH.
Permet à l'auto-entreprise d'aller ajouter du contenu dans le html sans avoir à passer par moi systématiquement. 
Si conflit, je corrige.
Si ajout trop important (modification du carrousel ou ajout d'une carte) c'est moi qui fait.
